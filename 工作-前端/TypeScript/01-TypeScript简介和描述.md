#### TypeScript 描述

1. 是 JavaScript 的一个超集，支持ESMAScript 6
2. 微软开发的自由和开源的编程语言
3. 适用于开发大型应用，可编译成 JavaScript 运行于任意浏览器上

#### TypeScript 特性

- 类型批注和编译时类型检查
- 类型推断
- 类型擦除
- 接口
- 枚举
- Mixin
- 泛型编程
- 名字空间
- 元组
- Await

#### 安装 TypeScript

```
npm install -g typescript
```

检查版本

```
tsc -v
```

可以通过 tsc 来执行 ` .ts` 类型文件，会生成一个 ` .js` 类型文件
